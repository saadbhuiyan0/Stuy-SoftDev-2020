Bootstrap

Bootstrap is a front-end framework for web development created at Twitter that later became open-source. It's a very popular framework utilized by sites such as NBA.com, Walmart.com, Target.com, and Bloomberg.com.

Bootstrap includes plenty of HTML and CSS based design templates for a variety of uses. It's great because it's pretty easy to learn, it allows for responsive web design, and it's compatible with a wide range of browsers. You can use Bootstrap to change the way elements on your page look or add completely new elements like forms, buttons, tables, navigation, etc. The list goes on.

BootstrapCDN

CDN stands for Content Delivery Network. You don't need to understand how that works but essentially it delivers web content to you (the end user). BootstrapCDN is the recommended CDN for Bootstrap. It's a public CDN that can load CSS, JavaScript and images remotely, from its servers. It's servers are very reliable and handle huge loads of traffic delivering more than 70 billion requests a month!

Getting Started with Bootstrap

To use Bootstrap's compiled CSS, we are going to use the BootstrapCDN. To do so, we have to include this line of code in the head of the HTML file (HTML5 doctype is required):
<link rel="stylesheet" href="https://stackpath.bootstrapcdn.com/bootstrap/4.3.1/css/bootstrap.min.css" integrity="sha384-ggOyR0iXCbMQv3Xipma34MD+dH/1fQ784/j6cY/iJTQUOhcWr7x9JvoRxT2MZw1T" crossorigin="anonymous">

Navigation Bar

That dark gray bar that always sticks to the top of your screen is this is the navigation bar. We made it so that it is always there on your screen but that's just one of many customization options. You can change the colors, the links, the brand, the logo, etc. You can even add a dropdown menu to it or a form like a search bar.

This is a Jumbotron

(like the first blue box)

It's basically a large box that you can put stuff in. It's a nice way to isolate and call attention to something (ex: welcoming users of a website) . It is not limited to a gray box, it can be anything from a color to an image to a video.

To create a jumbotron you use a <div> element with class .jumbotron.

P.S. You can add code elements like the ones in the last line with the <code> tag.
