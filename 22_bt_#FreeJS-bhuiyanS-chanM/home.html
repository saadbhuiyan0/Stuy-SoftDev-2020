Bootstrap


Bootstrap is a front-end framework for web development created at Twitter that later became open-source. It's a very popular framework utilized by sites such as NBA.com, Walmart.com, Target.com, and Bloomberg.com.

Bootstrap includes plenty of HTML and CSS based design templates for a variety of uses. It's great because it's pretty easy to learn, it allows for responsive web design, and it's compatible with a wide range of browsers and it works especially well for mobile devices. You can use Bootstrap to change the way elements on your page look or add completely new elements like forms, buttons, tables, navigation, etc. The list goes on.


BootstrapCDN


CDN stands for Content Delivery Network. You don't need to understand how that works but essentially it delivers web content to you (the end user). BootstrapCDN is the recommended CDN for Bootstrap. It's a public CDN that can load CSS, JavaScript and images remotely, from its servers. It's servers are very reliable and handle huge loads of traffic delivering more than 70 billion requests a month!



Getting Started with Bootstrap


Bootstrap uses HTML elements and CSS properties that require the HTML5 doctype so that must be included at the beginning (as well as the lang attribute). Meta tag for the proper charset should be in the head as well as the responsive viewport meta tag to ensure proper rendering and zoom touching of the site on all devices. The beginning of the HTML file should include these lines: 


					<!DOCTYPE html>
					<html lang="en">
						<head>
							<meta charset="utf-8">
							<meta name="viewport" content="width=device-width, initial-scale=1, shrink-to-fit=no">
						</head>


To use Bootstrap's compiled CSS, we are going to use the BootstrapCDN. To do so, we have to include this line of code in the head of the HTML file: 


					<link rel="stylesheet" href="https://stackpath.bootstrapcdn.com/bootstrap/4.3.1/css/bootstrap.min.css" integrity="sha384-ggOyR0iXCbMQv3Xipma34MD+dH/1fQ784/j6cY/iJTQUOhcWr7x9JvoRxT2MZw1T" crossorigin="anonymous">



Typography


Typography has to do with styling and appearance. Which is pretty important, we can't have an ugly looking website now can we? Bootstrap has a lot of nice features to help make the text on the site more visually appealing. It changes the way the already existing HTML header tags <h#> look to be more modern. It also display headings of which there are four classes to choose from - .display-1|2|3|4. There is also a <small> element which is used for secondary texts. Here are some other nice features you might chose to implement in your design:

  • Text Colors: Classes for text color using .text-.
  • Text Formatting: Classes for the text formatting using .text-.
  • Text Background Colors: Classes for the background color of the text using .bg-.
  • Font Styles: Classes for the font's style using .fonts-.
  • Code: Element used to add text as code using <code>.
  • Keyboard Input: Element used to indicate keyboard inputs using <kbd>.
  • Blockquote: Classes used to add a blockquote by implementing .blockquote to a <blockquote> element.



Utilities


Utility classes are used to apply CSS-like styling quickly and efficiently to a variety of elements. Here are some other utility classes you might chose to implement in your design:

  • Borders
  • Border Color
  • Border Radius
  • Float/Clearfix
  • Width/Height
  • Spacing
  • Shadows
We won't get into detail about each class as there are so many and they are highly customizable. You should definitely take a look at them and play around, see what you like and don't like - these might be very helpful later on.



Grids


Bootstrap uses a 12 column wide grid system built with flexbox. Not all 12 columns have to be used individually, they can range from 12 columns of width 1 to 1 column that has width 12. They also do not need have equal width, you can have a 1 column width 2 and another width 4 and the final width 6. The grid system is responsive and will rearrange depending on screen size.

There are five grid classes:

  • .col- for extra small devices.
  • .col-sm- for small devices.
  • .col-md- for medium devices.
  • .col-lg- for large devices.
  • .col-xl- for extra large devices.
Note: The classes scale up.

Navbars


A navbar is essentially a menu, it can be horizontal, or vertical, collapse, or extend, etc. It is created using the .navbar class with the responsive class right after .navbar-expand-sm|md|lg|xl. The navbar contains an <ul> element with class="navbar-nav" with content as <li> elements with class="nav-item" which can then have a link with in an <a> element with class="nav-link".

A very basic example of what this would like look put together in HTML is: 


					<nav class="navbar navbar-expand-sm">
						<ul class="navbar-nav">
							<li class="nav-item">
								<a class="nav-link">Link 1</a>
							</li>
							<li class="nav-item">
								<a class="nav-link">Link 2</a>
							</li>
						</ul>
					</nav>

That dark gray bar that always sticks to the top of your screen is this our navigation bar. We made it so that it is always there on your screen but that's just one of many customization options. You can change the colors, the links, the brand, the logo, etc. You can even add, pills, tabs, dropdown menus, or even a form like a search bar (using JavaScript and jQuery). In our case all we did was add a .navbar-brand class, change around the style, use anchors, and make it stick to the top with .sticky-top.

Bootstrap also has a .nav class and we're not sure what the difference is as it seems they can be used interchangeably. Once we are educated on the difference we will amend this section appropriately.



This is a Jumbotron


(like the first blue box)

It's basically a large box that you can put stuff in. It's a nice way to isolate and call attention to something (ex: welcoming users to a website). It is not limited to a gray box, it can be anything from a color to an image to a video.

To create a jumbotron you use a <div> element with class .jumbotron.

Media Objects


Media objects are a nice way of putting media together with content. There are many ways to incorporate them. One way is with the .media class containing media content in a .media-body class. They can be nested, or arranged, or formatted and styled in many different ways.
